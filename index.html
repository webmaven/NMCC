GitHub Authentication

How to create a secure Personal Access Token (PAT):
1. Go to your GitHub Fine-grained PAT Settings.
2. Set repository access to: Only select repositories → NMCC.
3. Under Permissions, select Repository permissions → Contents → change access to Read and Write.
4. Generate the token, paste it below, and click verify. It is kept secure inside local storage.
